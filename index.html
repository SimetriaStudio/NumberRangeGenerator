Output Range of Numbers

This is a micro tool that lets you create all the numbers spanning a starting number and an ending number. For example if you need all the numbers between 1 and 5, the output range would be: 1 2 3 4 5. Of course this is a very crude example, but imagine you need every number between 1 and 10,000...that would be pretty labor intensive, but not with this tool. Enjoy!

NOTE: The difference between the starting number and ending number cannot be greater than 15,000. Do increments of 10,000.
